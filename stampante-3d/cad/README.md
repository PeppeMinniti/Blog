## üí° Formati File

### STL (Standard per Stampa 3D)
- **Formato:** STL binario
- **Uso:** Stampa 3D diretta, visualizzazione 3D, slicer
- **Dimensione:** 19 KB - 565 KB per file
- **Modifica:** Importa in Blender, MeshMixer, FreeCAD
- **Pro:** Universale, supportato da tutti gli slicer
- **Contro:** Non parametrico (difficile da modificare)

### DWG (CAD Professionale)
- **Formato:** AutoCAD DWG
- **Uso:** Modifica parametrica, produzione CNC, documentazione tecnica
- **Software:** AutoCAD, FreeCAD (con ODA Converter)
- **Contenuto:** Geometria 3D solida completa
- **Pro:** Modificabile, quotature, layer, parametrico
- **Contro:** Formato proprietario, software costoso
